Software 3D renderer
Completed: 2022-01-31 Language: Odin Github link

A software 3d renderer I wrote in Odin following "Learn 3d graphics programming from scratch" course by Gustavo Pezzi.

Can draw textured models (exported as .obj) on the CPU by following steps similar to what GPUs do in shaders and internally. Has some extra features not covered in the course.
